Movie Recommendation System

A project to recommend movies based on user preferences using collaborative filtering and content-based filtering techniques.

Project Overview

This project aims to recommend movies to users based on their preferences. We use collaborative filtering and content-based filtering techniques to make accurate recommendations.

Technologies Used

  • Python
  • Pandas
  • NumPy
  • Scikit-learn
  • Surprise Library

Implementation

The implementation involves the following steps:

  1. Data Collection: We use the MovieLens dataset for this project.
  2. Data Preprocessing: Clean and preprocess the data for analysis.
  3. Model Training: Train collaborative filtering and content-based filtering models.
  4. Evaluation: Evaluate the models using metrics like RMSE.
  5. Deployment: Deploy the model as a web application using Flask.
